Welcome to XBMAN's Kodi Addons.

1: YouNow - Kodi Addon. for younow.com
2: LiveMe - Kodi Addon. for liveme.com

* Install the xbman's repo: via Zip File:
1: Download the zip file: repository.XBMAN-1.3.zip
2: Install the repo: System / Add-ons / Install from zip file: repository.XBMAN-1.3.zip
3: Install from repo: System / Add-ons / Install from repository:
  : xbman's addons repository: Video Add-ons: YouNow/LiveMe.

* Install the xbman's repo: via Kodi Source:
1: Go to: System / File Manager.
2: Select Add Source.
3: Enter: http://xbman.cu.cc/repo as the media source and click Done.
4: Enter: xbman repo as the name for the media source and click Ok.
5: Go to Home and select System / Add-ons.
6: Select Install from zip file.
7: Select xbman repo.
8: Select repository.XBMAN-1.2.zip. Repo is now installed.
9: Select install from repository.
10: Select Xbman's Addon Repository.
11: Select Video Add-ons.
12: Select YouNow and click Install.
* The addon can now be found in: Video -> Addons ->
